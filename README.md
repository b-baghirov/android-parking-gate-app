# Android Parking Gate App

This is a simple Android application developed with Kotlin.
The project demonstrates basic authentication and access control logic
for a parking gate system.

## Technologies
- Kotlin
- Android Studio
- Firebase Authentication

## Features
- User authentication
- Simulated gate open/close logic
- Simple and clean UI
